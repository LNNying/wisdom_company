# 获取入库记录接口
### 请求地址（POST）
/api/instock-order-detail/selectInstockOrderAllDetail

### 请求类型 :

| action            | method |
| ----------------- | ------ |
| InstockOrderDetailController | selectInstockOrderAllDetail |


### 请求参数JSON
~~~~json
{
	        "orderNo": "234234234234", // 业务单据编号
            "subOrderNo": "123123123123", // 子单编号
            "matCode": "10500305", // 物料编号
            "plantCode": null, // 工厂编号
            "priority": null, // 优先级
            "batchNum": null, // 批次号
            "speStkFlag": null, // 特殊库存标识
            "speStkNum": null, // 特殊库存编号
            "startExpectedDate": 1551776620000, // 预计入库开始时间
            "endExpectedDate": 1551776660000, // 预计入库结束时间
            "matQty": 4, // 物料数量
            "matUnit": "个", // 物料单位
            "cargoOwner": null, // 货主
            "startCreateTime": 1551776620000, // 创建开始时间
            "endCreateTime": 1551776660000, // 创建结束时间
            "inStorageNum": 4, // 已入库数量（入库时反写）
            "orderDetailStatus": "2" // 单据明细状态
}
~~~~
